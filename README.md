## 什么是Frame Set？

Frame Set是一个专门为影视创作者打造的视觉参考库。简单来说，它就像是一个巨大的电影画面搜索引擎，里面收录了超过35万帧高质量画面，来自2885部电影、5373个音乐视频和8445个广告。

这些画面都是精心截取的，不是随便截的屏幕截图。每一帧都代表了某个特定的构图、光线、色彩或者情绪。你可以用它来做导演分镜、广告提案、情绪板，或者单纯找找灵感。

## Frame Set vs 竞品：选哪个？

市面上类似的工具不少，最有名的是ShotDeck。简单对比一下：

**ShotDeck**
- 优势：由《小丑》摄影指导Lawrence Sher创立，专业性强，电影画面质量极高
- 劣势：只有电影，没有音乐视频和广告；价格稍贵（12.90欧元/月）

**Frame Set**
- 优势：电影+音乐视频+广告，覆盖面更广；价格相对友好；界面现代
- 劣势：电影画面的深度可能不如ShotDeck

**Film Grab**
- 优势：完全免费，精选画面质量不错
- 劣势：没有搜索功能，只能按电影浏览；数量有限

**Flim AI**
- 优势：有AI搜索，还能找动态片段
- 劣势：相对较新，内容库还在建设中

我的建议是：如果你主要做叙事电影，ShotDeck更专业。如果你做的项目涉及商业广告和音乐视频，Frame Set是更好的选择。如果预算有限，可以先用Film Grab免费版，配合Frame Set的10次免费搜索。

## 一些使用小技巧

用了一段时间Frame Set后，我总结了几个提高效率的小技巧：

1. **多用情绪词而不是具体物体**：与其搜"汽车"，不如搜"孤独的汽车"或"速度感"
2. **善用筛选器**：按照画面比例、色调、时段等筛选，能更快找到你要的
3. **建立主题收藏集**：与其为每个项目建收藏，不如按"光线"、"构图"、"色彩"等主题分类，以后更好复用
4. **用相似搜索扩展思路**：找到一张接近的画面后，用相似搜索能发现你没想到的可能性
5. **移动端随时收藏**：看电影或者在街上看到好的画面，立刻用手机搜索类似的保存下来
